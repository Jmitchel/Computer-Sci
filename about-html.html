About HTML & CSS

About HTML

HTML is a abbreviation for "HyperText Markup Langauge". HTML is used to create the text that is visible on the page in the form of paragraphs and titles. HTML was formed in 1992 by Tim Berners-Lee.

Tag Syntax What It Is Used For
Paragraph <p> This syntax is used if
you want to add a
paragraph to your
website.
Heading <h1-6> This syntax is used if
you want to give your
webpage a heading.
Link <a href="url"> This syntax is used
if you want to add
a link that takes you
to a different page in
your website.

About CSS

CSS is a abreviated version of "Cascading Style Sheet". CSS is basically what adds color to your website, it can add a color background to your website, even give your menu bar rounded edges and change the color of the text from balck to blue, red, green, etc. CSS was released to the world on December 17, 1996 by the World Wide Website Consortium.

Syntax What It Is Used For
color Changes the font color of a certain
background-color Changes the background color of your page
border Adds a border to something
