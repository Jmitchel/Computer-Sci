About HTML & CSS

About HTML

HTML is a abbreviation for "HyperText Markup Langauge". HTML is used to create the text that is visible on the page in the form of paragraphs and titles. HTML was formed in 1992 by Tim Berners-Lee. HTML is typically writin with the commands p, h1-6, ul and ol.

About CSS

CSS is a abreviated version of "Cascading Style Sheet". CSS is basically what adds color to your website, it can add a color background to your website, even give your menu bar rounded edges and change the color of the text from balck to blue, red, green, etc.
